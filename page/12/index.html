内向

2012-12-21
如何理解内向的人，内向与腼腆的区别在于，内向的人对社交是真的没有兴趣而不仅仅是害羞。

社交对于他们而言，是相当消耗精力的事情，而不像外向的人那样是获取活力的方式。

所以，请不要误会他们，以为他们冷漠不热情，他们只是不认为人与人之间的热情互动是必要的罢了。

更不要说：嘿，你太内向了，要外向一些。

因为每个人都有自己的生活方式与价值观，而内向与外向间不存在谁对谁错的区别。

那要怎样跟他们相处呢？

见到了跟他热情说声Hi，然后将他成路边的小石子——你虽看得见，但是不会去注视他。

那样，他将是最自在的，也最喜欢你这样的人了。

至此，我终于可以理直气壮地对闯进我生活然后指手画脚的王八蛋们说滚了。

2012-12-10

#! /usr/bin/env python

-- coding: utf-8 --

import termios, sys, os
TERMIOS = termios

def getch():
        fd = sys.stdin.fileno()
        old = termios.tcgetattr(fd)
        new = termios.tcgetattr(fd)
        new[3] = new[3] & ~TERMIOS.ICANON & ~TERMIOS.ECHO
        new[6][TERMIOS.VMIN] = 1
        new[6][TERMIOS.VTIME] = 0
        termios.tcsetattr(fd, TERMIOS.TCSANOW, new)
        c = None
        try:
                c = os.read(fd, 1)
        finally:
                termios.tcsetattr(fd, TERMIOS.TCSAFLUSH, old)
        return c

def passwd():
